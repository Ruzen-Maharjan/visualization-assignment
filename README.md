# Data Science Lab 1 – Visualization Report

## Description
This project analyses GDP and life expectancy data for four selected countries
using data from the World Bank.

## Data Source
World Bank Open Data: https://data.worldbank.org/

## Files in this Repository
- ds lab1.py – Data cleaning and visualization code
- Two datasets used for this report - gdp.csv and life_expectancy.csv from world bank open data.

## How to Run
Open the Jupyter notebook and run all cells in order.
